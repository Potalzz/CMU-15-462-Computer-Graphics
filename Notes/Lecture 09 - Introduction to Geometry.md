## 기하학의 의미
기하학은 모양, 크기, 패턴, 위치를 연구하는 학문.  
또는 길이, 각도 같은 측정 가능한 양이 정의되는 공간을 연구하는 학문이다.
컴퓨터 그래픽스 관점에서는 기하를 디지털 표현으로 인코딩하는 문제가 핵심.

## 기하를 기술하는 관점

같은 도형도 서로 다른 관점으로 기술 가능하다.

![](../assets/images/Pasted%20image%2020260303114028.png)
- 언어적 기술: 단위원 같은 명명 기반
- 암시적 기술: 조건을 만족하는 점들의 집합으로 기술함. ex) $x^2+y^2=1$ 형태
- 명시적 기술: 매개변수를 입력하면 점이 나오는 형태로 기술함. ex) $(\cos\theta,\sin\theta)$ 형태
- 동적 기술: 미분방정식의 해로 경로를 기술
- 이산적 근사: 다각형으로 근사하고 $n\to\infty$에서 원형에 접근하는 방식
- 대칭, 곡률, 단층 데이터 같은 속성 기반 기술

이 관점 차이는 컴퓨터에서의 표현 선택과 직접 연결됨.

## 디지털 기하 표현의 큰 분류
디지털 기하 표현은 크게 암시적 표현과 명시적 표현으로 나뉨.
한 가지 최선은 없고, 기하의 종류와 수행하려는 작업에 따라 적합성이 달라짐.  

## 암시적 표현

### 암시적 표현 정의
암시적 표현은 점을 직접 나열하지 않고, 관계식을 만족하는 점들의 집합으로 도형을 정의함.  
일반적으로 표면은 $f(x,y,z)=0$인 점들의 집합으로 정의됨.  
예로 단위 구는 $x^2+y^2+z^2=1$을 만족하는 모든 점의 집합

부호를 갖는 함수라면 내부/외부 판정도 가능함. 예로 $f(x,y,z)<0$을 내부로 두는 방식이 있음.

### 작업 관점 특징
암시적 표현은 내부/외부 판정이 쉬운 대신, 표면 위 점을 찾는 작업이 어려움.

- 샘플링 어려움: $f(x,y,z)=0$만 주어지면 표면 위의 한 점을 찾는 것부터 비용이 큼.
- 내부/외부 판정 쉬움: 주어진 점을 함수에 대입해 부호나 값을 확인하면 됨.

![](../assets/images/Pasted%20image%2020260303114617.png)
예로 $f(x,y,z)=x^2+y^2+z^2-1$에 대해 점 $\left(\frac34,\frac12,\frac14\right)$는  
$\frac{9}{16}+\frac{4}{16}+\frac{1}{16}=\frac78<1$이므로 단위 구 내부로 분류됨.  

### 대표적 암시적 기법

#### 대수적 곡면
표면을 $x,y,z$에 대한 다항식이 $0$이 되는 점들의 집합으로 정의.
단순한 형태는 표현 가능하지만, 복잡한 형태에 대한 다항식을 구성하기가 매우 어려움.  
![](../assets/images/Pasted%20image%2020260303114649.png)

#### 구성적 솔리드 기하 CSG

기본 도형을 불 연산으로 조합해 복잡한 형상을 구성함.
- 합집합
- 교집합
- 차집합

표현은 연산 트리 형태로 쌓는 방식이 일반적임.
![](../assets/images/Pasted%20image%2020260303114704.png)

#### 블로비 표면

불 연산 대신 부드러운 블렌딩으로 표면을 구성함.  
예로 점 $p$를 중심으로 한 커널을 $\phi_p(x)=e^{-\|x-p\|^2}$로 두고, 여러 커널 합 $f:=\phi_p+\phi_q$의 등가면을 취하는 방식이 있음.  
등가값 선택에 따라 분리된 덩어리가 결합되는 위상 변화가 자연스럽게 나타남.  
![](../assets/images/Pasted%20image%2020260303114749.png)

#### 거리 함수 기반 결합

거리 함수 $d(x)$는 공간의 점 $x$에서 물체까지의 최단거리를 준다.  
두 거리 함수 $d_1,d_2$를 결합해 합집합 같은 연산을 구현할 수 있음.

- 합집합 거리의 한 예: $f(x)=\min(d_1(x),d_2(x))$

결합 방식에 따라 결과 형상이 달라짐.  
![](../assets/images/Pasted%20image%2020260303114801.png)

#### 레벨셋 방법
![](../assets/images/Pasted%20image%2020260303114817.png)
닫힌 형식의 함수 대신, 격자에 샘플된 함수값을 저장하고 보간해 등가면을 얻는 방식임.  
보간된 값이 $0$이 되는 곳이 표면이 됨.

장점
- 텍스처처럼 형상을 직접적으로 제어하기 쉬움.

단점
- 격자 샘플링으로 인한 앨리어싱 문제가 생김.
- 3차원 격자 저장 비용이 큼. 표면을 포함하는 2차원 표면을 3차원 격자에 담으면 비용이 $O(n^3)$ 규모로 증가함.

활용 예
- 의료 데이터에서 일정한 조직 밀도 등가면 추출
- 유체 시뮬레이션에서 공기-액체 경계까지의 거리를 인코딩

![](../assets/images/Pasted%20image%2020260303114902.png)
저장 비용은 표면 주변의 좁은 띠만 저장하는 방식으로 줄일 수 있음.


#### 프랙탈

자기유사성과 다양한 스케일의 디테일을 갖는 형태를 기술하는 방식.
자연 현상 묘사에 유용하지만, 원하는 형태로 제어하기 어렵다는 단점이 크다.

![](../assets/images/Pasted%20image%2020260303115008.png)
만델브로 집합은 반복 $z\leftarrow z^2+c$에서 $\lvert z\rvert$가 유계로 남으면 점 $c$가 집합에 속한다고 정의함.
반복 횟수나 발산 속도에 따른 색칠로 구조를 시각화함.

### 암시적 표현 장단점

장점
- 표현이 간결할 수 있음.
- 내부/외부 판정이 쉬움. 예로 점을 대입해 $f(x,y,z)$ 부호를 확인하면 됨.
- 거리 질의가 쉬울 수 있음. 예로 표면까지 거리함수 기반 표현이 가능함.
- 단순한 형태는 정확한 표현이 가능함.
- 합쳐짐/분리 같은 위상 변화 처리에 유리함.

단점
- 표면 위 점들을 찾는 작업이 비쌈. 예로 그리기 위해 샘플을 뽑는 과정이 어려움.
- 복잡한 형태를 닫힌 형식으로 모델링하기 어려움.

## 명시적 표현

### 명시적 표현 정의

명시적 표현은 표면 위 점들이 직접 주어지거나, 매개변수를 넣으면 점이 계산되는 형태임.  
일반적으로 표면 매개화는 $f:\mathbb{R}^2\to\mathbb{R}^3,\ (u,v)\mapsto (x,y,z)$로 표현함.  
예로 구의 매개화는 다음과 같이 쓸 수 있음.

- $f(u,v)=\big(\cos u\sin v,\ \sin u\sin v,\ \cos v\big)$
- $0\le u<2\pi,\ 0\le v\le \pi$

![](../assets/images/Pasted%20image%2020260303115057.png)

### 작업 관점 특징

명시적 표현은 표면 샘플링이 쉬운 대신, 내부/외부 판정이 어려운 경향이 있음.

- 샘플링 쉬움: $u,v$에 값을 넣으면 표면 위 점을 바로 얻음.
- 내부/외부 판정 어려움: 주어진 3차원 점이 표면 내부인지 판단하려면 역문제 또는 별도 테스트가 필요함.

예로 토러스 매개화가 주어졌을 때, 점 $(1.96,-0.39,0.9)$의 내부 여부를 직접 판정하기가 까다로움.
![](../assets/images/Pasted%20image%2020260303115120.png)

### 대표적 명시적 기법

#### Point Cloud

가장 단순한 형태로 점 $(x,y,z)$의 목록을 저장함.  
법선, 색 같은 속성을 함께 저장하는 경우가 많음.

장점
- 어떤 기하도 샘플로 표현 가능함.
- 충분히 촘촘하면 그리기 쉬움.

단점
- 샘플이 부족한 영역의 보간이 어려움.
- 연결성 정보가 없어서 처리/시뮬레이션이 어려움.

![](../assets/images/Pasted%20image%2020260303115214.png)

#### 폴리곤 메시

정점과 폴리곤을 저장함. 폴리곤은 보통 삼각형 또는 사각형임.

장점
- 처리/시뮬레이션에 유리함.
- 적응적 샘플링이 가능함.

단점
- 자료구조가 복잡해짐.
- 이웃 관계가 불규칙해 구현이 어려울 수 있음.

![](../assets/images/Pasted%20image%2020260303115206.png)

#### 삼각형 메시와 바리센트릭 보간

삼각형 메시의 표준 저장 방식은 다음과 같음.

- 정점: 좌표 $(x,y,z)$ 리스트
- 삼각형: 정점 인덱스 $(i,j,k)$ 리스트

삼각형 내부의 점은 바리센트릭 좌표로 선형 보간해 정의함.
![](../assets/images/Pasted%20image%2020260303115255.png)

## 선형 보간에서 곡선 표현으로

### 기저 관점의 선형 보간

1차원 선형 보간은 두 값의 선형 결합으로 쓸 수 있음.  
이를 기저 함수의 선형 결합으로 해석하면, 더 다양한 기저를 통해 더 풍부한 곡선 표현으로 확장 가능함.  

### 베르슈타인 기저

선형 보간은 본질적으로 1차 다항식을 사용함.  
고차 다항식을 쓰면 더 유연해지며, 일반적 다항식 기저 대신 베르슈타인 기저를 사용함.

![](../assets/images/Pasted%20image%2020260303115317.png)
- $B_k^n(t)=\binom{n}{k} t^k(1-t)^{n-k}$
- $0\le t\le 1,\ k=0,\dots,n$

### 베지에 곡선

베지에 곡선은 베르슈타인 기저로 표현한 곡선임.
![](../assets/images/Pasted%20image%2020260303115422.png)
$n=1$이면 선분이며 $n=3$이면 3차 베지에 곡선임.

![](../assets/images/Pasted%20image%2020260303115352.png)

주요 성질
- 끝점을 보간함.
- 끝 구간에서 제어 다각형에 접함.
- 제어점들의 볼록껍질 내부에 포함됨.


차수를 계속 올리면 제어점이 많아지지만, 고차 베지에 곡선은 중간 제어점 보간이 약해 제어가 어려워짐.


### 조각별 베지에 곡선과 연속성

고차 단일 곡선 대신 여러 저차 베지에 곡선을 이어 붙이는 방식이 널리 쓰임.  
그래픽 편집 도구, 폰트, SVG 등에 사용되는 표준적 접근임.  
![](../assets/images/Pasted%20image%2020260303115719.png)

조각별 베지에를 형식적으로 쓰면 다음과 같음.

- $\gamma(u):=\eta_i\!\left(\frac{u-u_i}{u_{i+1}-u_i}\right),\quad u_i\le u<u_{i+1}$

매끄러운 곡선을 위해 위치 연속성과 접선 연속성이 필요함.

- 위치 연속성: 조각 경계에서 점이 일치함.
- 접선 연속성: 조각 경계에서 접선 방향이 일치함.

3차 베지에는 자유도가 충분해 접선 연속성을 구성하기 유리하지만, 더 낮은 차수에서는 일반적으로 제약이 큼.  
![](../assets/images/Pasted%20image%2020260303115731.png)

## 곡선에서 표면으로

### 텐서 곱

두 1차원 함수를 곱해 2차원 함수를 만든다.

- $(f\otimes g)(u,v):=f(u)g(v)$

곡선 기반 아이디어를 표면으로 확장하는 기본 도구임.  
![](../assets/images/Pasted%20image%2020260303115741.png)

### 베지에 패치

베지에 패치는 베르슈타인 기저의 텐서 곱들의 합으로 정의됨.

- $B_{i,j}^3(u,v):=B_i^3(u)B_j^3(v)$
- $S(u,v):=\sum_{i=0}^{3}\sum_{j=0}^{3} B_{i,j}^3(u,v)\,p_{ij}$

$p_{ij}$는 제어점 격자임.  
![](../assets/images/Pasted%20image%2020260303115803.png)

### 베지에 표면과 패치 연결 제약
여러 베지에 패치를 연결해 큰 표면을 만들 수 있음.
각 패치를 규칙적인 $(u,v)$ 격자로 샘플링하면 그리기는 쉬움.

문제는 연속성.
모든 정점에서 정확히 4개의 패치가 만나는 정규 연결성만 가정하면 제약이 과도해짐.
흥미로운 형태와 좋은 연속성을 얻으려면 다양한 연결성을 허용하는 패치가 필요함.

## 스플라인 패치와 NURBS

### 스플라인 패치 계열

베지에 패치의 연결 제약을 완화하기 위해 다양한 대안이 존재한다.
- NURBS
- Gregory
- 기타 스플라인 패치 계열

선택 기준은 자유도, 연속성, 편집 난이도, 평가 비용 등

### 유리 B-스플라인과 NURBS

베지에 곡선은 원뿔곡선을 정확히 표현하지 못하는 문제가 있음.  
해결은 동차 좌표에서 보간한 뒤 평면으로 다시 투영하는 방식이며, 결과를 유리 B-스플라인이라 부른다.
![](../assets/images/Pasted%20image%2020260303115930.png)

NURBS는 비균일 유리 B-스플라인임.

- 비균일: 노트가 임의 위치에 존재 가능함.
- 유리: 동차 좌표 기반이며 가중치가 존재함.
- B-스플라인: 조각별 다항 곡선 구조임.

가중치 $w$는 정점의 영향력을 조절함.  
![](../assets/images/Pasted%20image%2020260303120023.png)

### NURBS 표면

NURBS 곡선의 텐서 곱으로 패치를 만들고, 여러 패치를 모아 표면을 구성함.  
장점은 평가가 쉽고 원뿔곡선을 정확히 표현 가능하며 높은 연속성을 얻기 쉬운 점임.  
단점은 패치 잇기와 편집이 어려울 수 있다는 점임.  
![](../assets/images/Pasted%20image%2020260303120032.png)

## Subdivision

세분화는 제어 곡선 또는 제어 메시에서 시작해 반복적으로 분할하고, 새 정점 위치를 계산하는 방식.
반복을 통해 매끄러운 극한 곡선 또는 극한 표면에 수렴.
![](../assets/images/Pasted%20image%2020260303120112.png)

세분화는 점의 소속 여부를 테스트하는 방식이 아니므로, 일반적으로 명시적 표현으로 분류하는 관점이 강하다.

### Lane–Riesenfeld 예시

한 가지 세분화 규칙은 다음과 같음다.

![](../assets/images/Pasted%20image%2020260303120256.png)
- 각 에지의 중점을 삽입함.
- 파스칼 삼각형의 $k$번째 행을 합이 $1$이 되도록 정규화해 이웃 가중치로 사용함.
- 예로 $k=2$면 가중치가 $\left(\frac14,\frac12,\frac14\right)$임.
- 극한은 차수 $k+1$의 B-스플라인이 됨.

![](../assets/images/Pasted%20image%2020260303120306.png)

### Subdivision Surface

거친 폴리곤 메시인 컨트롤 케이지에서 시작해 각 요소를 세분화하고 국소 평균으로 정점을 갱신함.  
대표 규칙으로 캐트멀-클라크, 루프 등이 있음.

일반적으로 고려할 이슈는 다음과 같음.

- 보간인지, 근사인지
- 정점에서의 연속성

스플라인보다 모델링은 쉬운 편이지만, 점별 평가는 더 어렵다.  
실무에서 널리 사용되는 접근.

## 표현 선택 가이드

표현 선택은 기하의 성격과 작업 요구에 의해 결정된다.

- 내부/외부 판정, 거리 질의, 위상 변화가 핵심이면 암시적 표현이 유리함.
- 샘플링, 렌더링, 국소 편집이 핵심이면 명시적 표현이 유리함.
- 원뿔곡선의 정확 표현과 CAD 친화성이 필요하면 NURBS 계열이 유리함.
- 자유형 모델링 파이프라인에서 작업 효율이 중요하면 세분화 표면이 유리함.
- 실제 시스템에서는 표현 간 변환이 빈번함.

## 요약
- 기하 표현에는 단일한 최적해가 없음.
- 암시적 표현은 판정과 위상 변화에 강하고 샘플링이 약함.
- 명시적 표현은 샘플링과 모델링 워크플로에 강하고 판정이 약함.
- 메시, 스플라인, NURBS, 세분화는 서로 다른 제약과 장점을 가짐.
- 실무 파이프라인은 용도에 맞는 표현을 선택하고 필요 시 변환함.
